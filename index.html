City Weather Summary

Summary: Latitude vs. X

The purpose of this project was to create a visualization dashboard website using visualizations used in a previous assignment. Specifically the data extracted from the OpenWeatherMap API. The data referenced is an example of the data obtained over 500 cities.

Similarly, each of our datasets were analyzed using Matplotlib to plot the difference of weather amoungst different cities around the world. Factors we looked at included: temperature, cloudiness, wind speed, and humidity. This visualization dashboard hopefully provides insite on the data analysis.
